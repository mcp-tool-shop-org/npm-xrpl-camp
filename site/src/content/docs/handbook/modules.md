XRPL Camp walks you through the XRP Ledger step by step. Each module teaches one skill and produces a verifiable result on the XRPL Testnet.

## Module overview

### Wallet Creation

Your first module creates an XRPL wallet on the Testnet. You will learn:

- What a wallet address is and how keypairs work
- How to fund a Testnet wallet using the faucet
- How to check your balance

By the end, you have a funded wallet ready for transactions.

### Sending a Payment

With a funded wallet, you send your first XRP payment. This module covers:

- Constructing a payment transaction
- Signing and submitting to the ledger
- Understanding transaction fees and reserves

You will see your payment confirmed on the live Testnet.

### Transaction Verification

After sending a payment, you learn to verify it independently. This module teaches:

- Looking up a transaction by its hash
- Reading transaction metadata (delivered amount, ledger index, result code)
- Understanding what "validated" means on the XRPL

This is the core literacy skill -- knowing how to read the ledger yourself.

### Certificate

Complete all modules and XRPL Camp generates a certificate of completion. The certificate is:

- Tied to your training session
- Verifiable against the XRPL Testnet
- Proof that you executed real ledger operations

## Checking progress

At any point, check which modules you have completed:

```bash
npx @mcptoolshop/xrpl-camp status
```

## Time commitment

The full training takes about 10 minutes. Each module builds on the previous one, so they are designed to be completed in order in a single sitting.
